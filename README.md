# okcoin-terminal

使用golang 开发的 okcoin 行情 terminal 显示

安装：

```
go get -u github.com/gizak/termui
go get github.com/gorilla/websocket
```

运行：
```
go run okcoin.go
```

按 q键 退出程序， 按 p键 暂停 再次按 p键 继续运行
